# Excel Sales Analytics Dashboard

## Project Overview
This project analyzes Pizza  sales data using Microsoft Excel to generate business insights through interactive dashboards.

The goal of this project is to demonstrate data analytics skills including data cleaning, pivot table analysis, KPI calculations, and dashboard creation.

## Dataset Information

The dataset contains **48,620 sales records** with **12 columns** representing orders_id, Pizza_name_id, order_dates, and financial metrics.

### Key Columns
- pizza_id
- order_id
- pizza_name_id
- quantity
- order_date
- Order_time
- Unit_price
- total_price
- pizza_size
- pizza_category
- pizza_ingredients
- pizza_name


## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- KPI Calculations
- Interactive Dashboard Design

## Dashboard Features

The Excel dashboard provides the following analytics:

### Key Performance Indicators (KPIs)

- Total_Revenue
- Avg_Pizza_order
- Toal_pizza_sold
- Avg_order_value
- Avg_pizza_per_order


### Visualizations

- Daily Trend for Total Order
- Hourly Trend for Total Order
- % of Sales by Pizza Category
- % of Sales by Pizza Size
- Total Pizza By Pizza Category
- Top 5 Best Sellers By Total Pizzas sold
- Bottom 5 worst Sellers By Total Pizzas sold

### Business Insights

**Busiest Days & Times** 
- Days : Orders are highest on weekends, firday/Saturday evenings
- Times :There are maximum orders from 12:00 - 01:00 Pm & after 5:00 - 8:00 pm

**Sales by Category & Size**
- Category : Classic Category contributes to maximum sales & total orders
- Size : Large size pizza contribute to maximum sales

**Best & Worst Sellers**
- Best : Chicken Pizzas are top revenue generators
- Worst : the Brie Carre  is of the Bottom in both order and revenue
